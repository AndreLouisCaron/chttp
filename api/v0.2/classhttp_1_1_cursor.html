Detailed Description

Iterator for HTTP headers.

Note
The class uses a Java iterator style rather than the pointer syntax from the stnadard library. This has been done to map the C++ API very closely onto the native C API.

Recommended use:

http::Head head(4*1024);
// ...
// Do something that buffers HTTP headers.
// ...
// OK, we're ready to process HTTP headers.
http::Cursor cursor(head);
while (cursor.next())
{
if (cursor.field() == "Content-Length") {
// ...
}
}
