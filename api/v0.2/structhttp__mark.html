int http_head_commit ( http_mark *  self)

Restore the header invariants after a successful partial push.

Parameters
selfA value obtained using http_head_mark before the partial push operation began.
Returns
0 on failure (e.g. invariants are not respected), else non-zero.
Precondition
http_head_push_value just succeeded.
Postcondition
The buffer's invariants are restored. If the function returns 0, and the sequence of partial push operations is cancelled. If the function returns non-zero, self does not include the partial push operations.

Appends partial header data.

Parameters
self
fieldPart of the HTTP header name.
sizeNumber of valid bytes starting at field.
Returns
0 on failure (e.g. attempted to exceed the buffer capacity), else non-zero.
Precondition
The caller has acquired a mark using http_head_mark.
Postcondition
The buffer's invariants are broken, iteration using http_cursor is prohibited. A successful call to http_head_cancel or http_head_commit is required to restore the buffer invariants.
